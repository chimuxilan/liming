# 智学通 - 教务培训管理平台

## 快速启动

### 方式一：直接打开
双击 `index.html` 即可在浏览器中运行。

### 方式二：本地服务器（推荐）
```bash
# Python 3
python3 server.py

# 或者
python3 -m http.server 8080
```
然后访问 http://localhost:8080

### 方式三：Windows 用户
双击 `start.bat` 即可自动启动。

### 方式四：Mac/Linux 用户
```bash
chmod +x start.sh
./start.sh
```

## 功能模块
- 🏠 **首页** - 工作台仪表盘、待办课节、快捷操作
- 👥 **学员管理** - 学员列表、新增/编辑/删除、导入导出
- 📚 **课程管理** - 课程卡片、进度跟踪、创建/删除
- 📋 **班级列表** - 班级管理、关联课程和老师
- ✋ **上课点名** - ⭐ 核心功能
  - 待点名课节列表（卡片式展示）
  - 一键点名：出勤/迟到/缺勤/请假 四种状态
  - 实时出勤统计
  - 点名详情查看
  - 考勤数据导出 CSV
  - 首页待办课节快速入口
- 📅 **课表管理** - 周视图排课、课程/会议/考试分类
- 🏫 **教室管理** - 教室信息、状态管理
- 📝 **成绩管理** - 成绩录入、等级评定
- 📊 **考勤统计** - 出勤率报表
- 🎓 **老师管理** - 教师信息、状态管理
- 📈 **数据报表** - 营收趋势、招生统计
- ⚙️ **系统设置** - 机构信息配置

## 技术说明
- 纯前端实现，零依赖
- 数据持久化（localStorage）
- 支持主流浏览器（Chrome、Firefox、Edge、Safari）
- 响应式设计，支持平板和手机访问

## 上课点名功能说明

### 操作流程
1. 从侧边栏点击「上课点名」或首页「待办课节」进入
2. 选择待点名的课节，点击「开始点名」
3. 为每位学生标记出勤状态（出勤/迟到/缺勤/请假）
4. 点击「保存点名」完成

### 筛选功能
- 按日期筛选课节
- 按班级筛选
- 按老师筛选

### 数据导出
- 支持导出考勤记录为 CSV 格式
- 包含：上课时间、班级、老师、学员、出勤状态
