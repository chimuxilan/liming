# 智学通 - 教务培训管理平台

## 快速启动

### 方式一：直接打开
双击 `index.html` 即可在浏览器中运行。

### 方式二：本地服务器（推荐）
```bash
# Python 3
python3 server.py

# 或者
python3 -m http.server 8080
```
然后访问 http://localhost:8080

### 方式三：Windows 用户
双击 `start.bat` 即可自动启动。

### 方式四：Mac/Linux 用户
```bash
chmod +x start.sh
./start.sh
```

## 功能模块
- 📊 工作台（仪表盘）
- 👥 学员管理
- 📚 课程管理
- 📅 排课系统
- ✅ 考勤管理
- 📝 成绩管理
- 🎓 教师管理
- 🏫 教室管理
- 📈 数据报表
- ⚙️ 系统设置

## 技术说明
- 纯前端实现，零依赖
- 支持主流浏览器（Chrome、Firefox、Edge、Safari）
- 响应式设计，支持平板和手机访问
